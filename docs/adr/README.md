# Architecture Decision Records

Use this directory for architecture decisions that have meaningful trade-offs or long-term consequences.

## Naming

Use sequential filenames:

```text
0001-short-title.md
0002-short-title.md
```

## Template

```md
# ADR NNNN: Title

## Status

Proposed | Accepted | Superseded

## Context

What problem, constraint, or opportunity led to this decision?

## Decision

What did we decide?

## Consequences

What trade-offs, follow-up work, or constraints result from this decision?
```
